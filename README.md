# react-native-a11y-focus

Set a11y focus by ref

## Installation

```sh
npm install react-native-a11y-focus
```

## Usage


```js
import { A11yFocusView } from "react-native-a11y-focus";

// ...

<A11yFocusView color="tomato" />
```
